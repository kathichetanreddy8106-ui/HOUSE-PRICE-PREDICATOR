# House Price Prediction

This project predicts house prices using Machine Learning.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Features
- Predict price based on area, bedrooms, bathrooms
- Linear Regression model
- Data visualization

## How to Run
1. Install libraries
2. Run house_price_prediction.py

## Output
Predicts house price based on input features
